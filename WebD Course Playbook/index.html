3) '/' means the root. This is something to use when your site is published. When viewing your site locally do not use it. Ex: Use 'images/a.png' instead of '/images/a.png' locally

Use percentages in css attributes to make the elements dynamic (relative to the size of the device you are using)

Note that 100% == 16px == 1em. Using 'px' is not dynamic but '%' and em 'are'

Use the 'rem' instead of 'em' and '%' to ignore the font size of any parent being applied to that element

It is recommended to use 'px' for margin instead of any other unit
